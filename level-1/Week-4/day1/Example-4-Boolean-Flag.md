# Example 4: Using a Boolean Flag

```plaintext
START
    // Declare and initialize a boolean flag
    SET isLoggedIn TO false

    // Check if the user is logged in
    IF isLoggedIn THEN
        DISPLAY "Welcome back, user!"
    ELSE
        DISPLAY "Please log in."
    ENDIF
END
