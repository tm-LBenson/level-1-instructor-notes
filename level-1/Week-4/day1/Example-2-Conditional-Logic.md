# Example 2: Conditional Logic - Checking Age

```plaintext
START
    // Declare and initialize age
    SET age TO 16

    // Check if age is greater than or equal to 18
    IF age >= 18 THEN
        DISPLAY "You are an adult."
    ELSE
        DISPLAY "You are not an adult."
    ENDIF
END
